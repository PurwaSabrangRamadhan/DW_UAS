Cara Upload Program Ke Github Dengan Git di Windows

Dengan bantuan github, kami bisa mengakses repository atau program kamu dimana saja selama terhubung ke internet.

Apa kamu punya banyak program di komputer kamu dan hanya menyimpannya di localhost? Bagaimana jika kamu ingin mengaksesnya dari luar komputer kamu, dengan komputer kampus misalnya? Ribet kan? Nah kali ini Nyrat.ID akan mengajarkan kamu cara upload program kamu ke website yang namanya GitHub.

Dengan bantuan github, kami bisa mengakses repository atau program kamu dimana saja selama terhubung ke internet. Kamu juga bisa menemukan berbagai macam source code program terkenal yang memiliki lisensi terbuka (Open Source) seperti Linux, Android, dan banyak lagi. Kalau kamu mengupload program kamu keisni, kamu bisa mengajak temanmu atau bahkan orang lain untuk bekerja sama membangun program. Caranya uploadnya gimana? yuk, langsung ke tutorialnya...

Download Git

Apa itu git? Git adalah program dasar dari github, dengan git kamu bisa mengendalikan dan memanajemen versi dari program kamu. Kamu bisa mendownloadnya di situs resminya
https://git-scm.com/downloads
Setelah itu, kamu bisa memilih dan menginstallnya sesuai dengan sistem operasi yang kamu gunakan. Pada tutorial kali ini, kita akan menginstallnya di windows

Buat Akun Github

Sebelum membuat repository baru di github, kamu perlu buat akun dulu dengan cara mengisi form registrasi seperti username, email dan sebagainya. Kamu bisa mendaftar di website resminya atau pada link dibawah
https://github.com/join

Buat Repository

Setelah berhasil registrasi dan login ke akun github, kamu bisa membuat repository baru dengan mengkik tombol new di dasbor bagian kiri
Setelah itu kamu bisa mengisi
  • Nama repository
  • Deskripsi repository (opsional)
  • Memilih apakah repository kamu bersifat 
    • public (dapat dilihat siapa saja) atau
    • private (hanya akun kamu yang bisa melihatnya)
  • Menambahkan file readme.md, file ini biasanya digunakan sebagai panduan atau dokumentasi jika ada yang ingin menggunakan program kamu.
  • Menambahkan file .gitignore yang berfungsi untuk memilih file yang tidak ingin dilacak
  • Menambahkan lisensi ke program kamu
Setelah di isi, klik "create repository".

Inisialisasi Git

Setelah berhasil menginstall git pada langkap pertama, selanjutnya inisialisasikan program kamu dengan cara
  • Masuk ke folder program kamu
  • Klik kanan dibagian kosong dan pilih "Git Bash here", selain itu kamu juga bisa menggunakan git GUI, tapi di tutorial kali ini kita akan menggunakan git bash saja agar terbiasa dengan perintah command line
  • Setelah itu akan muncul terminal baru dan masukkan perintah
    git init

    dan akan muncul forder baru bernama .git


Push Repository

Setelah berhasil inisialisasi pada program kamu, selanjutnya masukkan perintah dibawah untuk memasukkan semua file program kamu beserta foldernya ke dalam track
git add .
atau
git add *

Setelah itu, commit program yang telah kamu add dengan perintah

git commit -m "commit pertama"

Pada bagian "commit pertama" dapat kamu ubah sesuai dengan yang kamu mau

Selanjutnya ketikkan perintah dibawah untuk mengubah ke branch main

git branch -M main

Lalu masukkan perintah dibawah sesuai dengan username dan repository yang kamu buat

git remote add origin https://github.com/[username github]/[nama repository].git

contoh

git remote add origin https://github.com/PurwaSabrangRamadhan/tugas2.git

setelah selesai, push program kamu dengan perintah

git push -u origin main

Saat pertama kali kamu mengupload program maka akan muncul login page github, pastikan kamu login dengan akun yang sama saat membuat repository.

Setelah semuanya berhasil, maka akan muncul output seperti ini

Selesai

https://github.com/[username]/[nama repository]
